# CS 351 Project

I will create a movie review website where people may view and leave reviews. The website will have a rating system out of 10 and will have a darker/clean theme to mimic a theater. People will be able to log in and have accounts so that the website will remember their reviews and have a name to connect the review to. Website will have a section showing movies where you can sort by highest rated, lowest rated, and recently added.
